<b>-nogui</b>
    runs the Service watcher without GUI (Graphical User Interface).

<b>-autorefresh:X</b>
    Starts the auto refresh. X sets the interval between the autorefreshes.
    If you do not select an interval number the programme starts with a default interval.
    Also arguments -autorefresh must be followed by a colon (:) and the refreshTime value.
                    
<b>-encrypt</b>
    Obfuscates the passwords of an unencrypted file.

<b>-conf:path</b>
    Loads a custom configuration file.
